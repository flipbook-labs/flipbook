# Story

A Story is a [ModuleScript](https://create.roblox.com/docs/reference/engine/classes/ModuleScript) with a `.story` extension that renders a single UI component in isolation. Flipbook discovers Stories within the `storyRoots` declared by a [[concepts/storybook|Storybook]].

See [[api/story-format|Story Format]] for the full module API.

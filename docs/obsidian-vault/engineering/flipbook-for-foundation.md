# Flipbook for Foundation

## Overview

Initially I wanted to adopt Flipbook in lua-apps to cast a wide net on adoption. However, I think it would be better in the short term to first adopt Flipbook in Foundation.

This will allow me to gain some gradual adoption from my team. This also has the nice benefit of giving us feedback on the plugin before rolling it out to everyone at the company.

## Milestones

| **Summary** | **Description** | Status |
| --- | --- | --- |
| Flipbook has near 1:1 compatibility with Developer Storybook | Story/Storybook format mainly |   |
| Built-in plugin updated to consume upstream changes | Ideally I also want to make some improvements to this flow so it’s easier to maintain. I want to offload most of the work we’re doing in the mirror repo to the upstream one itself. |   |
| Team testing | Enable the main plugin flag and start playing around with it on Foundation’s codebase |   |
| General availability | Roll out to the whole company. Post to common Luau channels about it |   |

# Flipbook Mounts Anywhere

Places Flipbook needs to mount:

* DockWidetPluginGui
Plugin
Built-in Plugin
* ScreenGui
3D Viewport
In-Experience

When `plugin` doesn't exist there will be no way to manage settings. Flipbook needs to be able to know where it is currently mounted so we can switch some UI behavior around

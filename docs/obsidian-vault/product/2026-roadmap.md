# 2026 Roadmap

## Q1

### ModuleLoader Refactor

**Status**: ==Done==

1. Support for require-by-string
2. Improvements to how brittle module loading is in Flipbook. Better error recovery
3. Collaborating with the folks that maintain Jest Roblox

### Vide Support

**Status**: On Hold

### Controls Revamp

**Status**: In Progress

4. New control types:  Color picker, Date picker, Range slider, Radio, Multi-select
5. Number-based controls can have constraints applied to them (min, max, int/float)
6. Support for all UI Labs controls for easy interop between plugins

### Embed in Experience

**Status**: Not Started

7. Export a storybook to an experience to share current implementations with team members
8. Accompanying GitHub action for auto-deployments

# Flipbook Internal Release Strategy

1. Built-in plugin added to Studio
    1. Folks can start using it now! Just need to build it with `git rbx` and set `FFlagEnableFlipbook` to true
2. I need at least 1-2 people onboarded to Flipbook. They don’t need to be functional, but they should be lightly familiar with the codebase and the architecture
    2. Include plugin, storyteller, module-loader, backend, and other things that wouldn’t be obvious
3. Make it clear that this is to succeed Developer Storybook but that the two will need to coexist while Flipbook gain’s adoption
4. Flipbook needs to be setup with OTA so I don’t have to wait so long to deploy changes

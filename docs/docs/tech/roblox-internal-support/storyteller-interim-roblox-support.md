# Storyteller Interim Roblox Support

I think it would be reasonable to commit to using Roblox internal story definition props in Storyteller

I can mark those specific props as unstable with the caveat that they could disappear at any point. But those props will likely exist forever and we'll build on top of them. They would become legacy migration props that map to the new ones

Then we can slowly migrate our storybooks in-house over to the new format gradually

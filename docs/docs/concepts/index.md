# Concepts

Core concepts that underpin how Flipbook works.

- [[concepts/story|Story]] — A ModuleScript that returns a renderable definition
- [[concepts/storybook|Storybook]] — Tells Flipbook where to discover stories and how to render them
- [[concepts/module-loader|ModuleLoader]] — The sandboxing layer that isolates story execution
- [[concepts/storyteller|Storyteller]] — The backbone library for story discovery and rendering
- [[concepts/react|React]] — Roblox's Luau port of React 17
- [[concepts/storybook-js|Storybook.js]] — The JavaScript library that inspired Flipbook

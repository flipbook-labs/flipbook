# Product

Flipbook is a storybook plugin for Roblox Studio that allows Creators and Builders (Roblox employees) to implement, iterate, and preview seamlessly across devices.

# Northstars

- Flipbook is the ubiquitous storybooking plugin for Roblox UI
- Flipbook is used daily by engineers at Roblox to develop the Universal App, in-experience UI, and built-in plugins
- Everyone using another storybook plugin can easily migrate to Flipbook
- Story loading is 99% flawless and recovers quickly when something goes wrong
- Bring UI from anywhere and preview it with Flipbook

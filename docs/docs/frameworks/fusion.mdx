# Fusion

[Fusion](https://elttob.uk/Fusion/0.3/) is a reactive UI library created and maintained by Daniel P H Fox. Flipbook supports Fusion v0.3 via the `packages` object. Pass in your installation of Fusion to your Storybook and all stories will be rendered with it.

Example project structure:

```
Project/
	Components/
		Button.luau
		Button.story.luau
	FusionStorybook.storybook.luau
```

Storybook:

<Tabs groupId="main">
	<TabItem value="default" label="Default" default>
		<CodeBlock className="language-lua" title="Project/Fusion.storybook.luau">
			{FusionStorybook}
		</CodeBlock>
	</TabItem>
	<TabItem value="storyteller" label="Storyteller">
		<CodeBlock className="language-lua" title="Project/Fusion.storybook.luau">
			{FusionStorybookStoryteller}
		</CodeBlock>
	</TabItem>
</Tabs>
